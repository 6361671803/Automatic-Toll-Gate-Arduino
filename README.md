AUTOMATIC TOLL GATE USING ARDUINO UNO

This project demonstrates an automatic toll gate system using Arduino UNO.
The gate opens and closes automatically based on vehicle distance.
video link of project:https://youtu.be/gPMiC5ysqVo?si=xUC-647sbjlTdqil

WORKING LOGIC
Distance less than 10 cm – Gate Opens
Distance greater than 10 cm – Gate Closes

COMPONENTS USED
Arduino UNO
Ultrasonic Sensor HC-SR04
Servo Motor SG90
Jumper Wires
Breadboard

CONNECTION DETAILS
Ultrasonic Sensor:
VCC – 5V
GND – GND
TRIG – D9
ECHO – D10

Servo Motor:
Signal – D6
VCC – 5V
GND – GND

APPLICATIONS
Automatic Toll Booths
Smart Parking Systems
Automatic Entry and Exit Gates
